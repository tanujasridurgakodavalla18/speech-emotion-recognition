# Speech Emotion Recognition

## Overview
This project detects human emotions from speech features using Machine Learning.

## Dataset
The project uses:
- Female_features.csv
- Male_features.csv

Due to GitHub file size limitations, the dataset files are not included in this repository.

## Model
Random Forest Classifier

## Results
- Accuracy: 87.32%
- Precision: 87.44%
- Recall: 87.32%
- F1 Score: 87.31%

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- Matplotlib
- Joblib
